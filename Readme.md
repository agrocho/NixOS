# NixOS Structure
## .dotfiles
These nix files are intended to be stored in your home directory in a `.dotfiles` folder `(~/.dotfiles/)`.

### hosts
These files describe the hostname of the intended endpoint (Toaster is my desktop, and Thunkpad is my laptop).

### modules
These files describe the different modules (gaming, packages (temporary until I unfuck the structure and make more individual flakes), etc).
